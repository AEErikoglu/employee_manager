A User can crate a workplace
if  User creates a workplace, he gets the role manager
the manager can invite employees to workplace

if a user accept an invitation, he gets added to the workplace
he gets the role Employee
he can enter days and hours that he is available for

manager can assign working ours for the days an hours the employee is available for
maanger can display when an Employee is awailable and works

- User can register/login with Azure authentication.
- User can create a workplace and automatically becomes manager.
- Manager can invite employees.
- Employee can accept invitation and join workplace.
- Employee can enter weekly availability.
- Manager can see employee availability.
- Manager can assign shifts only inside available times.
- Employee can see assigned working hours.
